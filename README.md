# Xplore-Careers
## Description
This is a quiz application that helps the youth find job placements in society in the area where they belong. The quizes analyse 4 courses in which a person may belong in respectively;
* Business
* Sciences
* Arts
* Engineering


#### Languages Used
* HTML5
* CSS3
* JAVASCRIPT
* JQUERY
